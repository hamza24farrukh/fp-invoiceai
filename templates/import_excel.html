Excel Import

Upload Excel files (.xlsx or .xls) to automatically extract supplier information. The system will identify unique suppliers with their:

  • Supplier Names (Transactors)
  • Transaction Types
  • Service Categories

Please ensure your Excel files have columns for transaction type, supplier name (transactor), and expense category. The system will automatically detect these columns.

Notes on Excel Structure

For best results, your Excel file should have:

  • Date information
  • Transaction type information (e.g., "EXPENSE", "PAYMENT", etc.)
  • Supplier/transactor names
  • Category information

The system will automatically attempt to identify these columns based on their headers. If column names aren't recognized, it will default to using:

  • Column 1 for dates
  • Column 2 for transaction types
  • Column 3 for supplier names (transactors)
  • Column 4 for expense categories
